# godot-template

This is a template for Godot projects. It includes a basic structure and some common files to help you get started quickly.

## Template Structure

```
.
├── .gitignore
├── .github
│   └── workflows
│       └── godot-ci.yml # GitHub Actions for CI
├── assets               # Binary assets
│   ├── fonts
│   ├── images
│   ├── models
|   └── music
│   ├── sounds
│   └── textures
├── materials
│   └── shaders
├── scenes               # Godot scenes
│   ├── game.tscn
│   └── main.tscn
├── scripts
│   ├── gamemanager.gd
│   └── main.gd
├── theme                # Theme files for UI
│   └── theme.tres
├── ui                   # User Interface components
│   ├── ...
│   └── menu.tscn
├── LICENSE
├── README.md
├── bus_layout.tres      # Bus layout for audio
└── project.godot
```
