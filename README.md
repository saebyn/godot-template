# godot-template

This is a template for Godot projects. It includes a basic structure and some common files to help you get started quickly.

## Template Structure

```
.
├── .gitignore
├── .github
│   └── workflows
│       └── godot-ci.yml
├── assets
│   ├── fonts
│   ├── images
│   ├── models
|   └── music
│   ├── sounds
│   └── textures
├── materials
│   └── shaders
├── scenes
│   └── main.tscn
├── scripts
│   ├── gamemanager.gd
│   └── main.gd
├── theme
│   └── theme.tres
├── ui
│   ├── ...
│   └── menu.tscn
├── LICENSE
├── README.md
└── project.godot
```
